## 运行前提

```bash
# 仓库根目录
python -m http.server 8080
```

- 需要 **Secure Context**（`localhost`），不要用 `file://`
- Chrome / Edge 94+，视频编码 H.264（avc1）

## CDN 依赖

| 库 | 用途 | 使用 Demo |
|----|------|-----------|
| [mp4box.js](https://github.com/gpac/mp4box.js) | MP4/fMP4 解封装 | 全部 |
| [mux.js](https://github.com/videojs/mux.js) | TS → fMP4 transmux | pipeline-visual |

## 管线概要

```
MP4 / M3U8 / TS
    → 解封装（mp4box / mux.js）
    → avcC → VideoDecoder.configure()
    → EncodedVideoChunk → decode()
    → VideoFrame → Canvas → close()
```
