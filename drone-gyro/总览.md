# Drone Gyro 主题总览

**无人机陀螺仪姿态** WebSocket 模拟：纯 Node.js 内置模块，60fps 二进制广播 pitch / roll / yaw，附浏览器测试客户端。

## Demo 索引

| 日期 | 文件夹 | 说明 | 运行方式 |
|------|--------|------|----------|
| 2025-05-31 | [2025-05-31-ws-broadcast](./2025-05-31-ws-broadcast/) | WebSocket 服务端 + 浏览器客户端 | 见下方 |

## 运行

### 1. 启动服务端

```bash
node drone-gyro/2025-05-31-ws-broadcast/server.js
```

默认地址：`ws://127.0.0.1:8765`

换端口：

```powershell
$env:WS_PORT=8766; node drone-gyro/2025-05-31-ws-broadcast/server.js
```

### 2. 打开客户端

```bash
python -m http.server 8080
```

浏览器：`http://localhost:8080/drone-gyro/2025-05-31-ws-broadcast/client-demo.html`

## 二进制帧格式（16 字节，小端）

| 偏移 | 类型 | 字段 |
|------|------|------|
| 0–3 | float32 | pitch 俯仰角（度） |
| 4–7 | float32 | roll 横滚角（度） |
| 8–11 | float32 | yaw 偏航角（度） |
| 12–15 | uint32 | timestamp（毫秒） |

广播间隔：**16ms**（约 60fps），含缓慢正弦漂移 + 随机抖动噪声。
