Part 0

一分钟看懂：抠像 = 按颜色「剪」视频

一句话：绿幕视频里，背景是浅绿色、甜甜圈是棕色。 程序对每个像素问：「你更像绿幕，还是更像甜甜圈？」 像绿幕 → 变透明；像甜甜圈 → 留下。

Demo 做的事：把 MP4 里的主体「剪」出来，变成 3D 世界里的一张透明卡片。

先认识 4 个词（后面会反复出现）

Key Color 键色
你要抠掉的颜色。当前视频背景是 #7C9170 浅绿，不是电视里的纯绿。
色差
某个像素的颜色，离 Key Color 有多远。越远越不像背景。
alpha 透明度
0 = 全透明（抠掉）；1 = 全不透明（保留）。边缘常在 0~1 之间。
Threshold / Slope
两个旋钮：决定「多像绿幕才算背景」和「边缘过渡有多软」。
Part 1

整体流程：每个像素都要过一遍关

Shader 对每一个像素独立判断。Threshold 和 Slope 就是这道判断题的「评分标准」。

Part 2

两个旋钮：Threshold 与 Slope

旋钮 A · Threshold 抠像阈值 — 「分界线画在哪」
比喻：一把剪刀。色差小于 Threshold → 当背景剪掉；大于 Threshold → 当主体留下。

Threshold 就是数轴上那条虚线：左边判背景，右边判主体。

Threshold 调大 vs 调小（看对比图）

旋钮 B · Slope 边缘斜率 — 「过渡带有多宽」
比喻：Ramp 斜坡 vs 悬崖。主体边缘的像素往往是「半绿半棕」，不可能非 0 即 1。 Slope 决定从「全透明」到「全不透明」之间，过渡有多宽、多柔。

Threshold 定「分界在哪」，Slope 定「分界附近有多宽的一条灰色地带」。

Slope 在数轴上的位置（和 Threshold 的关系）

Slope 调小 vs 调大（看对比图）

两个旋钮怎么配合？
你看到的問題像什么先调哪个
背景去不干净，有一圈绿甜甜圈外面包了一圈绿光↑ Threshold，或改 Key Color
甜甜圈边缘缺了一块主体被「啃」了↓ Threshold
边缘有锯齿、太硬像剪纸↑ Slope
边缘发虚、像雾主体轮廓糊↓ Slope

当前：Threshold=0.28，Slope=0.10 — 推荐起点，适合浅绿幕 #7C9170

推荐调参顺序（4 步）

  1. Key Color 设成 #7C9170（和背景一致）
  2. 只调 Threshold：直到绿边和缺角达到平衡
  3. 再调 Slope：把边缘锯齿修柔和，别太大
  4. 打开 Demo，Pan 运镜转一圈，看不同角度有没有绿边
