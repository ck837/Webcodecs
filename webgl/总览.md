## 运行

**Shader 沙盒**

```bash
cd webgl/2026-06-21-shader-learning-sandbox
npm install
npm run dev
```

浏览器打开 http://127.0.0.1:5173。

**电影质感视频增强器**

```bash
cd webgl/2026-06-21-cinematic-video-enhancer
npm install
npm run dev
```

浏览器打开 http://127.0.0.1:5174。可选：将 MP4 放入 `public/sample.mp4`。

**3D 空间 AIGC 元素调配器**

```bash
cd webgl/2026-06-21-aigc-3d-matting
npm install
npm run dev
```

浏览器打开 http://127.0.0.1:5175。推荐绿幕 MP4 放入 `public/greenscreen.mp4`。

教程：http://127.0.0.1:5175/tutorial.html

**教程：** http://127.0.0.1:5173/tutorial.html（或 Demo 左侧「完整教程」链接）

## 学习路径

```
src/shaders/vertexShader.ts    ← 顶点阶段：位置 → 屏幕
src/shaders/fragmentShader.ts  ← 像素阶段：改颜色（从这里玩起）
src/components/ShaderSphere.tsx ← useFrame 更新 uTime / uMousePosition
```
