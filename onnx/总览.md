## 一键运行

```bash
cd onnx/2026-06-15-mobilenet-classifier
python -m http.server 8080
```

打开 http://127.0.0.1:8080/index.html — 页面会自动加载示例耳机图并运行 Top-5 推理。

原理讲解 / 实验课：[lab.html](./2026-06-15-mobilenet-classifier/lab.html)（步步执行，ipconfig 式输出）

## 资源说明

| 文件 | 说明 |

| `sample/headphones.jpg` | 示例图片（已内置） |

## 缺失模型时

```powershell
cd onnx/2026-06-15-mobilenet-classifier
powershell -ExecutionPolicy Bypass -File scripts/download.ps1
```

## 注意事项

- 必须用 **HTTP 服务**，不能用 `file://`
- 首次运行从 CDN 加载 ORT WASM（约数 MB）
- 推荐 Chrome / Edge 最新版
