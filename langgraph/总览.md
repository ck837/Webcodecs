# LangGraph 主题总览

**LangGraph** = AI Agent 的**有向图状态机**（Nodes + Edges）。本主题聚焦「Agent 逻辑可视化」——Python 编译拓扑，Canvas / WebGL 渲染控制台。

## 三大框架对照（人话版）

| 框架 | 本质 | 可视化价值 |
|------|------|------------|
| **LangChain** | 线性 Pipeline（jQuery 胶水库） | 低 — 一条线，不需要拓扑图 |
| **LangGraph** | 有向图状态机（XState / React Flow） | **极高** — 节点/边/条件回流，完美喂给 Canvas |
| **AutoGen** | 多 Agent 聊天室 | 中 — 气泡对话流，偏文本低维 |

## 运行

```bash
cd langgraph/2026-06-01-visual-agent-topology
pip install -r requirements.txt
python visual_agent.py              # 跑 Agent + 导出 topology.json
python visual_agent.py --serve      # 导出 + 浏览器打开 Canvas 可视化
```
